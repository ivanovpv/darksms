Algorithms and libraries

  • DES cipher designed by IBM (http://en.wikipedia.org/wiki/Data_Encryption_Standard)
  • AES/Rijndael cipher designed by Vincent Rijmen and Joan Daemen (http://en.wikipedia.org/wiki/Advanced_Encryption_Standard)
  • Anubis cipher (Copyright (с) 2001, 2008) designed by Vincent Rijmen and Paulo S. L. M. Barreto (http://en.wikipedia.org/wiki/Anubis_(cipher))
  • SHA-1 cryptographic hash function designed by NSA (http://en.wikipedia.org/wiki/SHA-1)
  • SHA-2 set of cryptographic hash functions designed by NSA (http://en.wikipedia.org/wiki/SHA-2)
  • Whirlpool (Copyright (с) 2001, 2008) cryptographic hash function developed by Vincent Rijmen and Paulo S. L. M. Barreto (http://en.wikipedia.org/wiki/Whirlpool_(cryptography))
  • Elliptic Curve Cryptography designed by Neal Koblitz and Victor S. Miller (1985-1987) (http://en.wikipedia.org/wiki/Elliptic_curve_cryptography)
  • Diffie-Hellman public key cryptography designed by Whitfield Diffie and Martin Hellman (1976) (http://en.wikipedia.org/wiki/Diffie%E2%80%93Hellman_key_exchange)
  • R250 random number generator, designed by S. Kirkpatrick and E. Stoll - Journal of Computational Physics, 40, p. 517 (1981).
  • Software developed by the OpenSSL Project for use in the OpenSSL Toolkit. (http://www.openssl.org/) - see full license below
  • Cryptographic software written by Eric Young as part of OpenSSL - see license below
